== Frequently Asked Questions ==

= Is this plugin really free? =

Yes! A-Tables & Charts Lite is completely free with all essential features. We also offer a PRO version with advanced features like Excel import, charts, and more.

= What file formats are supported? =

The free version supports CSV files. Upgrade to PRO for Excel (XLS, XLSX), JSON, and XML support.

= Can I edit tables after creating them? =

Absolutely! You can add, edit, and delete rows and columns at any time using our intuitive table editor.

= Is it mobile responsive? =

Yes! All tables are fully responsive and work beautifully on mobile devices, tablets, and desktops.

= How do I display a table on my site? =

After creating a table, use the shortcode `[atable id="YOUR_TABLE_ID"]` in any post, page, or widget. You can find your table ID in the tables list.

= Can I customize the table appearance? =

Yes! You can customize pagination, search visibility, sorting options, and more through the display settings.

= Does it work with my theme? =

A-Tables & Charts works with any properly coded WordPress theme. The tables use standard WordPress styling and are designed to blend seamlessly.

= How many tables can I create? =

You can create unlimited tables! There are no restrictions on the number of tables.

= Can I export my tables? =

Yes! The free version supports CSV export. PRO users can also export to Excel and PDF formats.

= Will it slow down my site? =

No! The plugin is optimized for performance with caching and lazy loading. Tables load fast even with large datasets.

= What's the difference between free and PRO? =

The free version includes CSV import, basic editing, and frontend display. PRO adds Excel/JSON/XML import, interactive charts, advanced export options, and priority support.

= Do you offer refunds? =

Yes! We offer a 30-day money-back guarantee on all PRO purchases.

= How do I get support? =

Free users can use the WordPress.org support forums. PRO users get priority email support with responses within 24 hours on business days.
